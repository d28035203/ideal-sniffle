# Neon Tunnel

A **Three.js** neon tunnel: Catmull-Rom spline tube, floating wireframe boxes, bloom post-processing, and orbit controls.

## Features

- Spline-driven tube geometry
- Floating edge boxes along the path
- UnrealBloom-style glow
- Camera motion along the tunnel

## Tech

Three.js · EffectComposer / bloom · JavaScript

## Run

```bash
git clone https://github.com/d28035203/neon-tunnel.git
cd neon-tunnel
npx --yes serve .
```
